This program requires the a PDB file of the incorporated residue in the proper sugar pucker configuration (C3'-endo for A-RNA, C2'-endo for B-DNA)
Command line >> python RNA_mute.py -h for arguments and their description.

Input required:

PDB of system to be mutated
PDB of residue to be inserted
Residue ID of the residue to replace

Output:

PDB of the mutated system
